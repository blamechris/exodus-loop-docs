## Development

```bash
# Install dependencies
npm ci

# Local development
npx quartz build --serve

# Build for production
npx quartz build
```

## Deployment

Push to `main` branch triggers GitHub Actions workflow:
1. Build Quartz site
2. Deploy to gh-pages branch
3. Site available at https://blamechris.github.io/exodus-loop-docs/
